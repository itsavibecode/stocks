          🔑 Finnhub API Key
          
        

        
Required. Without your own Finnhub API key, news and prices will not fetch. Get a free key at finnhub.io/register. Key syncs with your signed-in account.

        

          
          
          
        

        

      


      

        

          🔒 Backup API Keys
          
        

        
Optional. If Finnhub fails or hits its quota, the app falls back to these in order. Both are free with sign-up.
